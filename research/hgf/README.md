HGF (Hierarchical Gaussian Filter)

Usage
- Entrypoint: `scripts/Problem Specific/Hierarchical Gaussian Filter/run_hgf.jl`
- Outputs saved to `results/<timestamp>/HGF/`.

Modules
- `Utils.jl`: parameters and synthetic data generation
- `Model.jl`: RxInfer models and constraints/meta
- `Viz.jl`: plotting helpers
- `Run.jl`: filter and smoothing runners
