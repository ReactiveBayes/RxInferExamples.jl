## Learning Paths

### Path 1: Quick Start (30 minutes)
1. Read [Quick Start](quickstart.md) - 5 min
2. Run an example - 5 min
3. Skim [Visualization Guide](visualization_guide.md) - 10 min
4. Explore output files - 10 min

### Path 2: Comprehensive (2-3 hours)
1. Read [Quick Start](quickstart.md) - 5 min
2. Read [Comprehensive Summary](comprehensive_summary.md) - 45 min
3. Study [Generic Agent Interface](generic_agent_interface.md) - 30 min
4. Study [API Reference](index.md) - 20 min
5. Review [Visualization Guide](visualization_guide.md) - 30 min
6. Run examples and experiment - 20 min

### Path 3: Developer (4+ hours)
1. All of Path 2
2. Deep dive [Generic Agent Interface](generic_agent_interface.md) - 30 min
3. Read [Implementation Complete](implementation_complete.md) - 60 min
4. Read [Enhancements Summary](enhancements_summary.md) - 30 min
5. Study source code - variable
6. Run test suite - 20 min
