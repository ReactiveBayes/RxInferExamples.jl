## Learning Paths

### Path 1: Quick Start (30 minutes)
1. Read [Quick Start](QUICKSTART.md) - 5 min
2. Run an example - 5 min
3. Skim [Visualization Guide](VISUALIZATION_GUIDE.md) - 10 min
4. Explore output files - 10 min

### Path 2: Comprehensive (2 hours)
1. Read [Quick Start](QUICKSTART.md) - 5 min
2. Read [Comprehensive Summary](COMPREHENSIVE_SUMMARY.md) - 45 min
3. Study [API Reference](index.md) - 20 min
4. Review [Visualization Guide](VISUALIZATION_GUIDE.md) - 30 min
5. Run examples and experiment - 20 min

### Path 3: Developer (4+ hours)
1. All of Path 2
2. Read [Implementation Complete](IMPLEMENTATION_COMPLETE.md) - 60 min
3. Read [Enhancements Summary](ENHANCEMENTS_SUMMARY.md) - 30 min
4. Study source code - variable
5. Run test suite - 20 min
